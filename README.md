## Back-end

### Opstarten

1. cd backend
2. Pnpm i
2. Pnpm create-env
3. Pnpm start OF pnpm dev

### Testen

1. cd backend
2. pnpm create-env
2. pnpm test

## Front-end

### Opstarten

1. cd frontend
2. pnpm create-env (.env aanmaken)
2. pnpm i (install dependecies)
3. pnpm dev

### Testen

1. cd frontend
2. pnpm create-env (.env aanmaken)
2. pnpm i (install dependecies)
3. pnpm test
